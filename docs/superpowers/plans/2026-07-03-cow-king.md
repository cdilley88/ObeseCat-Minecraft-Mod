# Cow King Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a dedicated `Cow King` entity that spawns once inside each `Cow King Fort` as a giant joke-boss placeholder, with a 5x visual scale and no combat tuning yet.

**Architecture:** Keep this as a custom cow-derived entity so we can reuse existing cow movement and animation while leaving the boss logic easy to extend later. Put the visual scale in a dedicated client renderer, keep the spawn logic in a fort-scoped server helper patterned after the bunker resident spawner, and keep the first pass intentionally non-hostile so we do not lock in combat behavior too early.

**Tech Stack:** Java 21, Minecraft 1.21.1, NeoForge 21.1.228, entity registration, client renderer registration, chunk-load spawn hooks, game tests, Gradle wrapper build pipeline.

- [ ] **Step 3: Add a red/green game test for single-spawn behavior**

Create `src/main/java/com/fende/obesecat/gametest/CowKingGameTests.java` with a test that calls the helper twice and asserts the fort still contains only one tagged Cow King.

Use this shape:

```java
@GameTestHolder(ObeseCatMod.MOD_ID)
@PrefixGameTestTemplate(false)
public final class CowKingGameTests {
    private static final String TEMPLATE = "village/plains/houses/cow_king_fort";

    @GameTest(template = TEMPLATE)
    public static void cowKingFortSpawnerSpawnsOnlyOneBoss(GameTestHelper helper) {
        BoundingBox box = new BoundingBox(0, 0, 0, 16, 16, 16);
        CowKingFortSpawner.ensureCowKing(helper.getLevel(), box);
        CowKingFortSpawner.ensureCowKing(helper.getLevel(), box);
        long count = helper.getLevel().getEntitiesOfClass(
                CowKing.class,
                new AABB(0, 0, 0, 17, 17, 17),
                entity -> entity.getTags().contains(CowKingFortSpawner.STRUCTURE_ENTITY_TAG)
        ).size();
        helper.assertTrue(count == 1, "Cow King Fort should only spawn one Cow King");
        helper.succeed();
    }
}
```
